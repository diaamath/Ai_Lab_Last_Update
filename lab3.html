Training Questions

Question 1: In a Normalizing Flow (NF) model, what mathematical feature must the mapping function f(z) satisfy?
Correct Answer: B (Bi-directional and perfectly invertible)

Explanation: Normalizing Flows require bijective (one-to-one) mapping functions so that latent variables can generate raw data coordinates, and vice-versa, with exact mathematical reconstruction.

Question 2: Given the forward function x = f(z) = 3z + 1, calculate the exact data space element x when the latent value z = 4.0.
Correct Answer: B (x = 13.0)

Explanation: Substituting z = 4 into the expression gives f(4) = 3(4) + 1 = 12 + 1 = 13.0.

Question 3: If an inverse network mapping yields z = f⁻¹(x) = (x - 1)/3, what is the value of z when processing data point x = 1.0?
Correct Answer: A (z = 0.0)

Explanation: Applying the formula yields: f⁻¹(1) = (1 - 1) / 3 = 0 / 3 = 0.0.
